# P2 XS Implementation Plan: Task Completion Notification + Idiomatic Scroll-to-Bottom

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Fire a VSCode notification when a stream completes and the chat panel is hidden; replace imperative scroll calls with a reactive Solid.js `createEffect`.

**Architecture:** `MessageHandler` gains an `onStreamEnd` callback field wired in `extension.ts`. `App.tsx` replaces 4 manual `scrollToBottom()` calls with one `createEffect` watching `messages()`.

**Tech Stack:** TypeScript, VSCode Extension API, Solid.js, Vitest

### Task 4: Replace manual scroll calls with `createEffect`

**Step 1: Add `createEffect` import**

On line 1 of `App.tsx`, add `createEffect` to the import:

**Step 2: Add the reactive effect**

Inside the `App` component, after the `scrollToBottom` helper (around line 74), add:

```ts

  chatStore.messages(); // subscribe to signal — fires on new messages and streaming chunks

    messagesEndRef?.scrollIntoView({ behavior: 'smooth' });
  });
});
```
